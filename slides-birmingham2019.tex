\begin{frame}{The mystery of nongeometric sequents}
  Let~$\TT$ be a geometric theory, for instance the theory of rings.

  % Illustration

  \textbf{Theorem.} There is a \hil{generic model}~$U_\TT$, a model such that
  for any geometric sequent~$\sigma$ the following notions coincide:
  \begin{enumerate}
    \item The sequent~$\sigma$ holds for~$U_\TT$.
    \item The sequent~$\sigma$ holds for any~$\TT$-model in any topos.
    \item The sequent~$\sigma$ is provable modulo~$\TT$.
  \end{enumerate}

  \justifying
  \textbf{Main observation.} The generic model can validate nongeometric
  sequents which are not provable modulo~$\TT$ and which do not hold for
  arbitrary models in arbitrary toposes.
\end{frame}
